webpack !

webpack 이란? :D

Webpack은 JavaScript 애플리케이션을 위한 정적 모듈 번들러입니다.

모듈 번들러란 프로젝트의 여러 개별적인 모듈들을 하나의 번들 파일로 결합하는 도구를 말합니다. 이는 웹 개발에서 여러 개의 JavaScript 파일, CSS 파일, 이미지 파일 등을 하나의 파일로 번들링하여 웹 애플리케이션의 성능을 향상시키는 데 도움이 됩니다.

로더는 웹팩이 모듈을 해석할 때 사용되며, 다양한 파일 형식을 처리하고 변환하는 역할을 합니다. 예를 들어, JavaScript 파일을 해석하기 위해 Babel 로더를 사용하여 ES6+ 코드를 ES5로 변환하거나, CSS 파일을 해석하기 위해 CSS 로더를 사용하여 CSS를 JavaScript로 변환할 수 있습니다. 이 외에도 이미지 파일, 폰트 파일 등 다양한 파일 형식을 처리하기 위한 로더들이 존재합니다.

webpack.config.js에 스크립트추가 

const path = require('path');
const HtmlWebpackPlugin = require('html-webpack-plugin');
const MiniCssExtractPlugin = require("mini-css-extract-plugin");

module.exports = {
    mode: 'production', //production,development
    entry: {
        index : './src/index.js'
    },
    output:{
        path: path.resolve(__dirname,'public'),
        filename: 'js/[name].js'
    },
    plugins: [
        new MiniCssExtractPlugin({
            filename: './css/style.css',   //번들후 css 분리위치
        }),
        new HtmlWebpackPlugin({
            template : './src/index.html', //번들전
            filename : './index.html',     //번들후
            hash: true,                    //해쉬설정
            showErrors: true               //에러설정
        })
    ],
    module: {
        rules: [
            {
                test: /\.css$/i,
                // use: ["style-loader", "css-loader"],
                use: [MiniCssExtractPlugin.loader, "css-loader"],
                exclude: '/node_modules'        //요 경로는 제외시킴
            },
        ],
    },
}

Vite가 무엇이니? :D

Vite.js는 Vite 플랫폼을 위한 공식 JavaScript 프레임워크입니다. Vite.js는 빠른 개발 속도와 효율적인 성능을 제공하기 위해 설계되었습니다.

1. hello world 출력하기

import React from "react";
import ReactDOM from "react-dom/client";

const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(<h1>Hello World</h1>);

//Hello World

2. JSX

JSX는 React에서 사용되는 자바스크립트 문법 확장입니다. JSX는 HTML과 유사한 구문을 사용하여 UI 구성 요소를 만들 수 있게 해줍니다.

import React from "react";
import ReactDOM from "react-dom/client";

const name = "getgrovy";

const hello = <h1>hellow {name}</h1>;;

const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(hello);

//Hello getgrovy

함수를 사용해서 출력

import React from "react";
import ReactDOM from "react-dom/client";

function helloName(){
    return name.nick
}

const name = {
    nick : "getgrovy",
}

const hello = <h1>Hello, {helloName()}</h1>;
const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(hello);
//Hello getgrovy
