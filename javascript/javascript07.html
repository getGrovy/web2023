01. Math.random()

0~1 사이의 난수 값을 반환합니다

번호 기본값 메서드 리턴값 클릭 
{
    const arrNum1 = [100, 200, 300, 400, 500];
    let text1 = arrNum1.join('');
    let text2 = arrNum1.join(' ');
    let text3 = arrNum1.join('*');
    console.log(arrNum1);
    console.log(text1);
    console.log(text2);
    console.log(text3);
}
